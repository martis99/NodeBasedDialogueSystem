Welcome to the documentation for the Node-\/\+Based \doxylink{namespace_dialogue}{Dialogue} System for Unity! This system is designed to streamline and enhance the creation of interactive dialogues in your Unity projects. Whether you are developing an RPG, a visual novel, or any game requiring complex narrative interactions, our system offers an intuitive and powerful solution.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_features_sec}

\begin{DoxyItemize}
\item {\bfseries{Intuitive Node Editor}}\+: Create and manage dialogue trees using a visual, drag-\/and-\/drop interface.
\item {\bfseries{Branching Dialogues}}\+: Easily design dialogues with multiple choices and branching paths.
\item {\bfseries{Event Triggering}}\+: When question changes it generates events.
\item {\bfseries{Nodes grouping}}\+: Ability to group dialogues for easier management
\item {\bfseries{Save and Load System}}\+: Implement saving and loading of dialogue states for continuous gameplay.
\item {\bfseries{Undo/redo}}\+: Ability to Unode or redo action for easy editing
\end{DoxyItemize}\hypertarget{index_getting_started_sec}{}\doxysection{\texorpdfstring{Getting Started}{Getting Started}}\label{index_getting_started_sec}
To start using the Node-\/\+Based \doxylink{namespace_dialogue}{Dialogue} System in your Unity project, follow these steps\+:


\begin{DoxyEnumerate}
\item {\bfseries{Import the Package}}\+: Add the package to your Unity project via the Package Manager.
\item {\bfseries{Open the \doxylink{namespace_dialogue}{Dialogue} Editor}}\+: Navigate to {\ttfamily Window -\/\texorpdfstring{$>$}{>} \doxylink{namespace_dialogue}{Dialogue} -\/\texorpdfstring{$>$}{>} Graph} to open the dialogue editor.
\item {\bfseries{Create a \doxylink{namespace_dialogue}{Dialogue} Tree}}\+: Create scritable object ({\ttfamily Assets -\/\texorpdfstring{$>$}{>} Create -\/\texorpdfstring{$>$}{>} \doxylink{namespace_dialogue}{Dialogue} Graph}) and use the editor to connect dialogue nodes.
\item {\bfseries{Integrate with Your Game}}\+: Attach created scritable object to the {\ttfamily \doxylink{class_dialogue_manager}{Dialogue\+Manager}}.
\end{DoxyEnumerate}\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_usage_sec}
